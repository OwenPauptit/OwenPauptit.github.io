C#

I decided in March 2020, that C# would be the most suitable language for my A level coursework.

I started off by creating some simple text-based programs that run in the console. By May, I picked up xamarin for android and made a few apps.

Then in July I began creating webapps, and started my coursework.

Graph Algorithms

After watching MIT Open CourseWare Mathematics for Computer Science lectures on Graph Theory, I decided to implement some of the algorithms I learned into a webapp. The website allows the user to create and move nodes, and construct weighted edges between nodes. The algorithms currently implemented are
  • Greedy colouring algorithm
  • Random spanning tree algorithm
  • Greedy minimum weight spanning tree algorithm
The webapp can be found here

Chemical Molecule Drawing

I modified my graph algorithm code to display chemical molecules. The user can add a variety of atoms and create bonds between them.

Bonds can be created by selected "Add Bond", then clicking on two different atoms.

Additional bonds between the same atoms (i.e. double bonds, triple bonds etc.) can be added in the same way.

The webapp can be found here

Dijktra's Shortest Path Algorithm

Dijkstra's Shortest Path Algorithm is a method of finding for finding the shortest paths between nodes in a graph.

In this implementation, however, I use a matrix, where each node is two █ characters in the console.

The program will find the shortest path from a start point to an end point while avoiding obstcales (walls).

Notes App

Notes!" is a note taking app that lets you create new notes, save those notes, and edit existing notes created with the app. Each note has a title and body.

The app works by saving text files with the title of each note being the name of the text file (i.e. "title.txt") and the content of the note saved inside the document.

Dungeon Explorer

Dungeon Explorer is a simple game played in the console.

The player navigates (WASD) through a series of procedurally generated rooms, encountering a variety of enemy types which can be killed (Arrow Keys).

The goal is to kill the most enemies as possible before dying
